## 🧵 第三篇：极客硬核贴 (Geek Style)

**主题**：Don't Trust, Verify. 为什么我选择自己手撸一个 Crypto 插件？💻

**(1/6)**
市面上的看盘插件很多，但我都不敢用。
原因很简单：**闭源 = 黑盒**。 ⬛️
你不知道它后面有没有偷偷上传你的剪贴板（私钥），或者在后台挖矿。
本着 "Don't Trust, Verify" 的 Web3 精神，我花了周末两天，用 **React + Vite** 手撸了一个纯前端、无后端的 Chrome 扩展。

**(2/6)**
🛠️ **硬核技术栈**

- **Core**: Manifest V3 (跟进最新标准)
- **UI**: React 18 + TailwindCSS (Glassmorphism 设计)
- **Data**: Direct WebSocket (直接连 Binance/OKX 交易所接口，**无任何中间服务器**转发)
- **State**: Chrome Storage API (本地化存储)
极简架构，内存占用极低，绝对不卡浏览器。🚀

**(3/6)**
⚡️ **性能优化 (Performance)**
为了处理高频 WebSocket 推送（tick 级别），我做了特殊的 Throttling（节流）处理。

- 图标 Badge 更新频率控制在 1s 一次，避免 UI 闪烁。
- 报警逻辑在 Service Worker 后台运行，即使关闭 Popup 也能实时监控。
- 只有你在看它的时候，K 线图才会懒加载 (Lazy Load)。

**(4/6)**
🔒 **Privacy First**
代码完全开源。
你可以去审计我的 `background.js`，你会发现：
❌ 没有 Google Analytics
❌ 没有 Sentry 报错上传
❌ 没有第三方 API 请求
它只是一个纯粹的、跑在你本地的 WebSocket 客户端。
这就叫 **Code is Law**。

**(5/6)**
🎨 **UI/UX**
虽然是工具，但审美不能妥协。
抛弃了传统的 Bootstrap 风格，全部重写了一套 **Dark Mode** 优先的组件库。
支持一键 Pin 住窗口，写代码的时候把窗口挂在屏幕角落，既能盯盘又不影响工作流。
